# Hunting Communication App

React Native mobile app for hunters to share locations and communicate in remote areas.

## Features
- Real-time GPS location sharing
- Emergency alerts with location
- Group messaging
- Offline-first design

## Tech Stack
- React Native + Expo
- Node.js + Express backend
- Socket.IO for real-time communication
- PostgreSQL database

## Setup

```bash
# Backend
cd backend
npm install
npm run dev

# Mobile app (new terminal)
cd mobile-app
npm install
npx expo start
```
